# Vemplate 使用说明

![操作](./operaiton.gif)

## 功能

* 快速创建`vue2`、`vue3`常用模板的`vue`文件
* 默认`css`预处理语言为`scss`

## 使用

1. 右键文件夹，选择 Vemplate
2. 输入文件名称
3. 选择模板，按下回车键即可成功创建带有模板的`vue`文件
